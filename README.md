# LioraLog

**Research Progress Tracker**

Track your research progress and collaborate with your team seamlessly. 

## Deployment
This project is currently deployed on Firebase Hosting.

## Tech Stack
- React + Vite
- TypeScript
- Tailwind CSS + shadcn/ui
- Firebase (Hosting, Firestore)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/itzRR/LioraLog.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```
